# dymock (dynamic mock)

A Python-based service to generate mock APIs dynamically from OpenAPI (Swagger) specifications. This tool streamlines API development by allowing developers to mock APIs with customizable responses, simulate latency, and test frontend-backend interactions effortlessly.

-> Check openapi specification 
Example use in this repo come from 
https://learn.openapis.org/examples/v3.0/petstore-expanded.html
